Para iniciar o algoritmo:
Você deve estar dentro do direório "Paxos". (cd /path/to/file)
Dê comando: python main.py

No arquivo main.py são instanciados os objetos Proposers, Acceptors e Leaner
A funcao que inicia o algoritmo é a função "start", da classe "Server", que deve ser chamada depois de todos os objetos
serem instanciados, e começará a determinar o Paxos.
